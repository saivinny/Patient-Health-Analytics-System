# Patient Health Analytics System

The Patient Health Analytics System is a Python-based healthcare analytics project designed to analyze patient health records and identify important medical insights such as stroke risk, lifestyle patterns, and health conditions.
The project applies Exploratory Data Analysis (EDA) techniques on a large healthcare dataset containing 171,999 patient records and 22 health-related features.

Hospitals and healthcare organizations generate large amounts of patient data every day, making manual analysis difficult and time-consuming. This system helps automate the analysis process and supports data-driven healthcare decisions.

## Project Objectives

* Understand the structure and quality of healthcare data
* Clean and preprocess patient records
* Analyze distributions of medical and lifestyle factors
* Identify relationships between health indicators and stroke occurrence
* Generate insights that can support preventive healthcare strategies

## Technologies Used

* Python
* Object-Oriented Programming (OOP)
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Dataset Features

The dataset includes:

* Demographics (age, gender, marital status, etc.)
* Lifestyle information (smoking status, physical activity, sleep hours)
* Clinical measurements (glucose level, BMI, hypertension, heart disease)
* Stroke outcome data

## Key Analysis Performed

* Data cleaning and preprocessing
* Handling missing values
* Exploratory Data Analysis (EDA)
* Correlation analysis
* Visualization of health trends and stroke risk factors

## Importance of the Project

According to the NHS, cardiovascular diseases and strokes are among the leading causes of death worldwide. By analyzing patient health patterns, this project helps identify high-risk individuals early and supports preventive healthcare measures.

## Outcome

The project successfully demonstrates how healthcare data can be analyzed using Python to discover meaningful insights and improve understanding of stroke-related risk factors.
